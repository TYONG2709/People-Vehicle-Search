Welcome to POWER SEARCH

This is a page where you can search the information of the drivers and vehicles, either by looking up with driver's name or the vehicle directly.

People Search can be used to find the driver's information such as Address, DOB (Date Of Birth), License Number and License's Expiry Date just by entering details - their name or driving license number.
→ Go to this page!

Vehicle Search have the similar functions but it will return vehicle's model, owner's name, license number etc. You can look up these information by entering the vehicle's registration (plate) number.
→ Go to this page!

You can Add a Vehicle by providing the registration (plate) number, make, model, and the colour of the vehicle, and its owner (If owner was not added in the system, you have to provide owner's information too).
→ Go to this page!
